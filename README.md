## ✨ Features

### 📝 Create Time Capsules
- Write messages to your future self
- Add title and category
- Save instantly

### 🔒 Unlock Date System
- Choose a future unlock date
- Capsules remain locked until that date

### ⏳ Live Countdown Timer
- Days
- Hours
- Minutes
- Seconds

### 📂 Memory Archive
- View unlocked memories
- Automatically move unlocked capsules to archive

### 🔎 Search Functionality
- Search by title
- Search by category
- Search memories instantly

### 🌙 Dark Mode
- Modern UI
- Persistent theme settings

### 💾 Local Storage
- No backend required
- Saves capsules permanently in browser

### 📱 Responsive Design
- Mobile
- Tablet
- Desktop

## 🎯 How It Works

### Step 1
Create a time capsule.

### Step 2
Write a message.

### Step 3
Choose an unlock date.

### Step 4
Save the capsule.

### Step 5
Wait until the unlock date arrives.

### Step 6
Read your message from the past.
